# TRADEOFFS

## SQLite vs PostgreSQL
SQLite chosen for simplicity during development.
PostgreSQL better for production scalability.

## Django Templates vs React
Django templates used for faster implementation.
React would provide better frontend interactivity.

## Local Deployment vs Cloud
Local deployment easier for testing.
Cloud deployment needed for public access.

## Suspicious Detection Logic
Simple threshold-based detection implemented.
Advanced ML detection could improve accuracy.
